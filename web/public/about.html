About AnExplorer

AnExplorer File Manager (File Explorer) is an Another File Manager which helps you manage your files easily. It is designed for all android devices including Phones, Phablets, Tablets and Android TV. It's a fully designed with Material guidelines by Google. It supports all Android versions from Jellybean, Kitkat, Marshmallow to Nougat. With it you can easily find, browse, move, compress, and otherwise manage your apps easily. You can Explore and manager File and Folders on Internal storage / External Storage / USB Storage and Free RAM on the device. File Manager helps you to backup your apps (apk) of Tools and Games so you can restore them again without downloading. File Manager makes it easy to share files over bluetooth or other social networking apps. If you have any questions, suggestions, or comments, email us at support@dworks.io.
